The Elix project seeks to provide a comprehensive set of solid, well-designed web components that implement very common user interface patterns. Each component can be used as is, or as the foundation for new components.

See the main Elix repository for details.

Demos

For now, this placeholder site just hosts demos of the project's top-level components. See all demos here.

  • Alert dialog. A modal dialog presenting buttons for a small set of possible responses.
  • Autosize textarea A text area that expands to contain its text.
  • Dialog. A standard modal dialog.
  • Drawer. A slide-out drawer useful for presenting navigation and other commands at small screen sizes.
  • Responsive drawer. A demo that makes use of a Drawer at small (mobile) screen sizes.
  • List box. Shows how several mixins cooperate to enable a very basic single-selection list box.
  • Horizontal list. A list box with a horizontal orientation.
  • Browser plugin list. A list box with a hard-coded set of list items.
  • Popup. A lightweight popup that appears over other page elements.
  • Sliding carousel Lets the user navigate laterally through a sequence of child elements.
  • Tabs. A set of panels with classic rounded tabs.
  • Toast. A small popup that slides in to present a short message.
  • Toolbar tabs. Tabs used as the overall navigation UI model for a mobile app.

Components with a prominent horizontal axis or focus on direction are designed to flip their logic in right-to-left languages like Arabic and Hebrew. Those components have demo variations that exercise their right-to-left support, including Carousel, Drawer, Tabs, and Toast.

The project also includes an extensive series of web component mixins and auxiliary components you can use to create your own custom elements. The mixins are used in the component demos above, but you can also see some mixins used on their own:

  • FocusVisibleMixin. Shows a focus ring on an element only when focused with the keyboard.
  • SingleSelectionMixin. Tracks a single selected item in a list.
  • TouchpadSwipeMixin and TrackpadSwipeMixin. Handle swipe gestures on touch screen and trackpads respectively.
